# klibcpp

klibcpp (kedixa's personal c++ toy library)

# Content

- timer: 计时器
- multiarray: 多维数组
- unsigned_bigint: 无符号大整数类
- bigint: 带符号大整数类
- rational: 有理数类

# Compile
```bash
make build # build library
make check # run tests
make rpm   # build rpm package
make clean # remove all build directories
```
